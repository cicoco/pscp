# Intro
Scp command with parameters.

# How to use
```
$ npm install pscp --save
```

```
pscp host username password file1 file2... RemoteDestination

```

Demo:    
```
pscp 10.22.30.1 root root test.txt /test/    

pscp 10.22.30.1 root root test.txt hello.json /test/    

```
